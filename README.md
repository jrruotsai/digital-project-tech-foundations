# 企業應用與數位專案技術基礎

**Technical Foundations for Digital Projects**

這是一套寫給參與企業系統與數位專案工作者的技術基礎課程。課程從系統運作、網路、資料、API 與身分安全，一路延伸到部署、問題排查及 AI 協作，協助學習者逐步建立參與技術專案所需要的共同語言與基礎能力。

## 課程目標

這套課程希望幫助學習者：

- 理解企業系統、網站、資料庫與整合服務如何共同運作。
- 聽懂技術團隊在討論的名詞、架構、限制與風險。
- 看懂資料如何在不同系統之間流動，以及 API、Webhook、SDK 等工具扮演的角色。
- 遇到系統異常時，知道應先確認哪些資訊、如何描述問題，以及應與誰溝通。
- 建立與工程師、供應商及專案利害關係人合作時所需的技術判斷基礎。

## 適合對象

本課程適合沒有資訊工程背景，但工作上需要參與企業系統或數位專案的人員，例如：

- 企業應用顧問、導入顧問與系統顧問。
- 專案經理、產品經理、Business Analyst 與 Customer Success。
- 數位行銷、CRM、MarTech、電商與營運人員。
- 經常需要與工程師、IT 團隊或軟體供應商合作的人員。
- 希望建立完整技術基礎，而不只是零散記憶專有名詞的學習者。

不需要具備程式設計經驗。內容以概念理解、實務情境與跨團隊溝通為主。

## 課程內容

課程涵蓋以下四個階段：

1. 資訊系統的共同語言
2. 資料與系統整合
3. 應用程式與身分安全
4. 系統交付、問題分析與 AI 協作

每一章都是獨立的小單元，可以依序閱讀，也可以從目前最需要的主題開始。章節中包含簡短檢核，方便確認是否掌握核心概念。學習進度只會儲存在目前裝置的瀏覽器中，不會上傳或收集個人資料。
